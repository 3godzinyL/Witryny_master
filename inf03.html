PODSTAWA 1: Co to jest zmienna?
Zmienna to "pudełko" w którym przechowujemy wartość.
Możemy tam włożyć tekst, liczbę, lub cokolwiek innego.
// Tworzymy zmienną o nazwie "imie" let imie = "Jan"; // Tworzymy zmienną "wiek" let wiek = 25; // Wyświetlamy alert(imie); // Pokaże: Jan alert(wiek); // Pokaże: 25
let - słowo do tworzenia zmiennej
imie - nazwa zmiennej
= "Jan" - przypisanie wartości
PODSTAWA 2: Typy danych
W zmiennych możemy przechowywać różne typy danych
// TEKST (string) let imie = "Jan"; let nazwisko = "Kowalski"; // LICZBA (number) let wiek = 25; let cena = 19.99; // PRAWDA/FAŁSZ (boolean) let czyPelnoletni = true; let czyZalogowany = false;
PODSTAWA 3: Łączenie tekstów
let imie = "Jan"; let wiek = 25; // Łączymy używając znaku + let zdanie = "Witaj " + imie + ", masz " + wiek + " lat"; alert(zdanie); // Wyświetli: Witaj Jan, masz 25 lat
Znak + łączy teksty w jeden
PODSTAWA 4: Dodawanie liczb
let a = 5; let b = 3; // Dodajemy liczby let suma = a + b; alert("5 + 3 = " + suma); // Wyświetli: 5 + 3 = 8
PODSTAWA 5: Co to jest funkcja?
Funkcja to "przepis" który możemy wykonać kiedy chcemy.
Zapisujemy kod raz, a możemy go używać wielokrotnie.
// Tworzymy funkcję function powitaj() { alert("Witaj!"); } // Wywołujemy (uruchamiamy) funkcję powitaj();
function powitaj() { ... } - tworzymy funkcję
powitaj() - wywołujemy funkcję (nawiasy!)
PODSTAWA 6: Funkcja z parametrem
Funkcja może przyjmować parametry - wartości które jej przekazujemy
// Funkcja przyjmująca parametr "imie" function powitaj(imie) { alert("Witaj " + imie + "!"); } // Wywołujemy z różnymi wartościami powitaj("Jan"); // Wyświetli: Witaj Jan! powitaj("Anna"); // Wyświetli: Witaj Anna!
PRZYKŁAD 1: Prosta zmienna
// Tworzymy zmienną let imie = "Jan"; // Wyświetlamy w alert alert(imie); // Pokaże: Jan
Wyjaśnienie:
1. Tworzymy zmienną "imie" i wpisujemy "Jan"
2. alert(imie) pokazuje zawartość zmiennej w okienku
PRZYKŁAD 2: Łączenie tekstu
let imie = "Jan"; let wiek = 25; // Łączymy teksty znakiem + alert("Witaj " + imie + ", masz " + wiek + " lat"); // Pokaże: Witaj Jan, masz 25 lat
Wyjaśnienie:
Używamy znaku + aby połączyć teksty i zmienne w jeden tekst
PRZYKŁAD 3: Dodawanie liczb
let a = 5; let b = 3; let suma = a + b; alert("5 + 3 = " + suma); // Pokaże: 5 + 3 = 8
Wyjaśnienie:
Tworzymy 3 zmienne: a, b, i suma. Do sumy zapisujemy wynik a + b
PRZYKŁAD 4: Funkcja - co to jest?
Funkcja to "przepis" na jakąś czynność.
Tworzymy ją raz, a potem możemy wywoływać wielokrotnie.
// Tworzymy funkcję o nazwie "powitaj" function powitaj() { alert("Witaj!"); } // Wywołujemy funkcję (uruchamiamy ją) powitaj();
Wyjaśnienie:
1. function powitaj() { ... } - tworzymy funkcję
2. powitaj() - wywołujemy funkcję (nawiasy są ważne!)

SQL - ZAPYTANIA DO BAZY DANYCH

⚠️ WAŻNE NA EGZAMINIE! BĘDZIESZ WPISYWAŁ W phpMyAdmin!
SELECT * - Pobierz wszystko
Pobiera wszystkie kolumny i wszystkie wiersze z tabeli
SELECT * FROM osoby;
SELECT - Wybrane kolumny
Pobiera tylko wybrane kolumny
SELECT imie, nazwisko FROM osoby; SELECT imie, wiek, miasto FROM osoby;
WHERE - Filtrowanie (liczby)
Pobiera tylko wiersze spełniające warunek
-- Osoby starsze niż 30 lat SELECT * FROM osoby WHERE wiek > 30; -- Osoby mające dokładnie 25 lat SELECT * FROM osoby WHERE wiek = 25; -- Osoby młodsze niż 18 lat SELECT * FROM osoby WHERE wiek < 18; -- Osoby w wieku 18-30 lat SELECT * FROM osoby WHERE wiek >= 18 AND wiek <= 30;
WHERE - Filtrowanie (tekst)
-- Osoby o imieniu Jan SELECT * FROM osoby WHERE imie = 'Jan'; -- Osoby z Warszawy SELECT * FROM osoby WHERE miasto = 'Warszawa'; -- Osoby których imię zaczyna się na 'A' SELECT * FROM osoby WHERE imie LIKE 'A%';
WHERE - Funkcje daty (YEAR)
⚠️ CZĘSTO NA EGZAMINIE!
-- Osoby urodzone po roku 2000 SELECT imie, nazwisko FROM osoby WHERE YEAR(dataUr) > 2000; -- Osoby urodzone w 1995 roku SELECT * FROM osoby WHERE YEAR(dataUr) = 1995; -- Osoby urodzone w latach 1990-2000 SELECT * FROM osoby WHERE YEAR(dataUr) >= 1990 AND YEAR(dataUr) <= 2000;
ORDER BY - Sortowanie
-- Sortowanie rosnące (od najmniejszego) SELECT * FROM osoby ORDER BY wiek ASC; -- Sortowanie malejące (od największego) SELECT * FROM osoby ORDER BY wiek DESC; -- Sortowanie alfabetyczne SELECT * FROM osoby ORDER BY nazwisko ASC; -- Sortowanie po kilku kolumnach SELECT * FROM osoby ORDER BY miasto ASC, nazwisko ASC;
DISTINCT - Unikalne wartości
-- Lista unikalnych miast (bez powtórzeń) SELECT DISTINCT miasto FROM osoby; -- Unikalne miasta posortowane SELECT DISTINCT miasto FROM adresy ORDER BY miasto ASC;
COUNT - Liczenie
⚠️ WAŻNE!
-- Ile jest osób w tabeli? SELECT COUNT(*) FROM osoby; -- Ile osób ma więcej niż 30 lat? SELECT COUNT(*) FROM osoby WHERE wiek > 30; -- Zliczanie z nazwą kolumny SELECT COUNT(*) AS liczba_osob FROM osoby;
SUM, AVG, MIN, MAX - Funkcje agregujące
-- Suma wszystkich wieków SELECT SUM(wiek) FROM osoby; -- Średnia wieku SELECT AVG(wiek) FROM osoby; -- Najmłodsza osoba SELECT MIN(wiek) FROM osoby; -- Najstarsza osoba SELECT MAX(wiek) FROM osoby;
JOIN - Łączenie tabel
⚠️ TRUDNE ALE WAŻNE!
-- Pobierz osoby i ich numery telefonów SELECT osoby.imie, osoby.nazwisko, telefony.numer FROM osoby JOIN telefony ON osoby.id = telefony.Osoby_id; -- To samo co wyżej (INNER JOIN) SELECT osoby.imie, telefony.numer FROM osoby INNER JOIN telefony ON osoby.id = telefony.osoby_id;
Wyjaśnienie:
- JOIN łączy dwie tabele
- ON określa jak je łączyć (po jakich kolumnach)
- osoby.id = telefony.Osoby_id - klucz obcy
LEFT JOIN - Wszystkie z lewej tabeli
-- Wszystkie osoby + ich telefony (jeśli mają) SELECT osoby.imie, telefony.numer FROM osoby LEFT JOIN telefony ON osoby.id = telefony.Osoby_id;
LEFT JOIN pokaże wszystkie rekordy z lewej tabeli (osoby),
nawet jeśli nie mają dopasowania w prawej tabeli (telefony)
WHERE z AND i OR
-- Osoby z Warszawy starsze niż 25 lat SELECT * FROM osoby WHERE miasto = 'Warszawa' AND wiek > 25; -- Osoby z Warszawy LUB Krakowa SELECT * FROM osoby WHERE miasto = 'Warszawa' OR miasto = 'Krakow'; -- Osoby z Warszawy lub starsze niż 40 lat SELECT * FROM osoby WHERE miasto = 'Warszawa' OR wiek > 40;
LIKE - Wyszukiwanie wzorców
-- Imiona zaczynające się na 'A' SELECT * FROM osoby WHERE imie LIKE 'A%'; -- Imiona kończące się na 'a' SELECT * FROM osoby WHERE imie LIKE '%a'; -- Imiona zawierające 'an' SELECT * FROM osoby WHERE imie LIKE '%an%'; -- Dokładnie 3 litery, zaczyna się na 'J' SELECT * FROM osoby WHERE imie LIKE 'J__';
% - dowolna ilość znaków
_ - dokładnie jeden znak
BETWEEN - Zakres
-- Osoby w wieku 20-30 lat SELECT * FROM osoby WHERE wiek BETWEEN 20 AND 30; -- Osoby urodzone między 1990 a 2000 rokiem SELECT * FROM osoby WHERE YEAR(dataUr) BETWEEN 1990 AND 2000;
IN - Wiele wartości
-- Osoby z Warszawy, Krakowa lub Gdańska SELECT * FROM osoby WHERE miasto IN ('Warszawa', 'Krakow', 'Gdansk'); -- Osoby mające 20, 25 lub 30 lat SELECT * FROM osoby WHERE wiek IN (20, 25, 30);
LIMIT - Ograniczenie wyników
-- Tylko pierwsze 5 osób SELECT * FROM osoby LIMIT 5; -- 10 najstarszych osób SELECT * FROM osoby ORDER BY wiek DESC LIMIT 10; -- 5 osób z największą pensją SELECT * FROM pracownicy ORDER BY pensja DESC LIMIT 5;
GROUP BY - Grupowanie
-- Ile osób jest w każdym mieście? SELECT miasto, COUNT(*) AS liczba FROM osoby GROUP BY miasto; -- Średni wiek w każdym mieście SELECT miasto, AVG(wiek) AS sredni_wiek FROM osoby GROUP BY miasto;
ALTER TABLE - Modyfikacja struktury
-- Dodaj nową kolumnę ALTER TABLE adresy ADD COLUMN numerMieszkania INT AFTER numer; -- Usuń kolumnę ALTER TABLE adresy DROP COLUMN numerMieszkania; -- Zmień typ kolumny ALTER TABLE osoby MODIFY COLUMN wiek INT;
